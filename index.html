hexo环境搭建

发表于 2017-01-18

创建Github域名

1.注册一个GitHub账号，用户名username,域名就是username.github.io
2.创建仓库，仓库名称就是username.github.io

安装

1.安装brew 

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2.安装git 

$ brew install git

3.安装Node.js

这里先安装nvm，是Node.js的版本管理器，这里使用brew安装

$ brew install nvm  
$ mkdir ~/.nvm  
$ export NVM_DIR=~/.nvm  
$ . $(brew --prefix nvm)/nvm.sh

安装完成后，重启终端并执行 

$ nvm install 4

4.安装Hexo 

$ sudo npm install hexo-cli -g

博客

1.创建博客 

$ hexo init username.github.io

2.主题安装
主题的选择可以去官网，这我选择next主题为例 

$ cd username.github.io
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

3.配置主题
基本配置
打开文件位置username.github.io/_config.yml,修改其中键值对，详细设置信息，这边只是简单地必要设置 

title: XXXX    //你博客的名字  
author: XXXX  //你的名字  
language: zh-Hans    //语言 中文  
theme: next   //刚刚安装的主题名称  
deploy:
   type: git    //使用Git 发布
   repo: https://github.com/username/username.github.io.git    // 刚创建的Github仓库

主题配置
主题配置文件在username.github.io/themes/next/_config.yml中修改，具体设置详情去不同主题的网址去找

4.写文章
把写好的md文件放到username.github.io/source/_posts目录下

5.测试
输入$ hexo s ，注意还是在username.github.io目录下，测试服务器启动之后，通过http://localhost:4000就可以访问了

6.安装部署发布工具 

$ npm install hexo-deployer-git --save

7.发布
测试没问题之后，就可以把文件上传到Github pages中 

$ hexo clean && hexo g && hexo d

上传成功之后就可以在浏览器中输入http://username.github.io，就能访问了

遇到的问题

在每次重启终端之后，输入nvm，就会提示-bash: nvm: command not found，解决这个问题的方法

1.先确保nvm安装成功
2.打开终端输入$ brew info nvm，会出现提示

3.这时候根据提示先在终端输入mkdir ~/.nvm
