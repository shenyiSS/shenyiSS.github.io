iOS 在应用内连接 WIFI

在 iOS11 之后，苹果在 NetworkExtension.framework 里面添加了 App 中直接登录 WIFI 的功能。这个 NetworkExtension.framework 里面有关于 VPN，WIFI 管理，DNS 代理等功能。

实现步骤

  1. 在项目中引入 NetworkExtension.framework，通过 build Phass -> Link Binay With Libraries 来引入
  2. 在 Capabilities 中找到 hotspot Configuration 的开关并打开
  3. 在代码中引入 import NetworkExtension
  4. 创建热点配置实例 let hotspotConfig = NEHotspotConfiguration(ssid: SSID, passphrase: password, isWEP: false)
  5. 开始连接

注意事项

仅支持 iOS11 以上版本，这种已知 WIFI 的 SSID 和密码的情况下，无需向苹果特别申请 Hotspot Helper APIs，直接用

WIFI 万能钥匙应用应该就是用的这个功能，先是搜索到附近 WIFI 的 SSID，再根据定位或者用户信息进行匹配在服务器查询到密码，这样就能连接到 WIFI 上了
