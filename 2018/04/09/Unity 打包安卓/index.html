Unity 打包安卓

整体实现思路就是先在Unity里面实现安卓的逻辑，保证方法名称的一致性，先把Untiy中导出安卓项目，然后再在安卓项目下打包整理逻辑

  1. build的时候切换成安卓，在play setting里面的Package Name这里设置BundleID,最上面的Product Name设置项目名称
  2. Target API Level 选择22，Minimum API Level选择16
  3. 在Build Setting弹出框里面选择Export Project，点击Export
  4. 导出项目成功之后，用AS打开项目，File->New->import Module，选择安卓工程师给的lib项目，点击Finish导入成功
  5. 打开Project（ProductName）->java->com.ehoo.pet(PackageName)->UnityPlayerActivity.java，@Override protected void onCreate(Bundle savedInstanceState)方法里面写初始化代码，并在此java文件里面添加需要安卓原生实现的逻辑，提供给外部接口

  6. 在Unity的脚本中调用安卓方法示例
