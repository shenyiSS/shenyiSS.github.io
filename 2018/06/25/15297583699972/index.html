黑苹果踩坑过程

准备

电脑配置

  • 主板 技嘉B250M-DS3H-CF
  • 处理器 i5 7500
  • 内存 威刚 DDR4 2400MHz 16GB
  • 硬盘 英睿达 MX300 525G
  • 显卡 NVIDIA GeForce GTX 1060 6GB 影驰
  • 声卡 realtek ALC887
  • 网卡 realtek RTL8186/8111/8112

其他准备

  • 白苹果电脑
  • 一个容量至少16G的U盘
  • 一个macOS可以识别U盘

下载系统安装包

按照教程

按照教程

设置BIOS

网上搜索了一下主板型号，找到一篇教程

进bios先恢复到默认，再按下列设置
1.BIOS Tab
Windows 8/10 Features = Other OS
Storage Boot Option Control = UEFI

2.Peripherals Tab
XHCI Handoff = Enabled (In USB Configuration)

3.Chipset Tab
VT-D = Disabled
IOAPIC 24-119 Entries = Disabled

安装系统

按照教程步骤

下面是我自己总结的步骤

  1. 从U盘启动，选择Boot macOS Install from Install macS High Sierra
  2. 抹掉(给新硬盘改名High Sierra)，选语言，重启
  3. 再从U盘启动，选择Boot macOS Install from High Sierra
  4. 会再重启，再从U盘启动，然后会出来安装进度条
  5. 再重启，再从U盘启动，选择 Boot macOS from High Sierra
  6. 然后会选择地区，下面一直设置系统，下一步
  7. 然后就能进入桌面了

安装驱动

按照教程步骤，根据自己的配置在MultiBeast选择需要安装的驱动

中间遇到两个问题

显卡问题

  1. 安装NVIDIA的WebDriver的驱动，参考链接
  2. 查看关于本机中的图形卡，检查一下有没有正确安装，如果没有正确显示可以去设置里面NVIDIA选项去切换成独立显卡

声卡问题

  1. 通过Multieast安装VooDooHDA万能驱动
  2. 在Clover的config页面，选Audio里面1写成11，勾选ResetHDA
  3. 进入macOS就能看到识别到声卡了

剩余问题

基本功能都能实现，暂时没有发现其他问题，可能后期还要把声卡驱动更新一下（AppleALC），USB的速度还没有测试

在tonymacx86上有好多配置的帖子，可以作为参考
