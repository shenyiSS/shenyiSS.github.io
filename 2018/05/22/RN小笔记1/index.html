RN小笔记1

  1. 正确打开姿势 进入到项目目录 open -a webstorm xxx/
  2. flexDirection的默认值是column而不是row，而flex也只能指定一个数字值
  3. 开发中cmd+r来reload，但是不知道我的电脑的cmd+r热键被哪个程序占用了，却无法定位这个程序。一番查阅后发现cmd+ctrl+z可以呼出调试菜单，菜单中有reload选项，按cmd+ctrl+z发现呼出QQ，于是在QQ设置里面删除了热键，就能实现呼出菜单->reload了。
  4. 在集成RN到已有项目时，用到了cocoapods，使用了pod update，出现了Failed to connect to GitHub to update the CocoaPods/Specs specs repo的提示，解决办法。
