组件

  1. 组件类的名称首字母大写，驼峰
  2. 使用React.createClass创建
  3. 每个组件类都必须实现render方法，返回值：null,false,组件模板
  4. 组件类只能包含一个顶层标签

props

props是组件自身的属性，一般用于嵌套的内外层组件中，负责传递信息（通常是由父层组件向子层组件传递）
props对象中的属性和组件的属性一一对应(children例外，不是跟组件的属性对应的，表示组件的所有子节点)，不要直接去修改props中属性的值
…this.props会复制父组件的所有属性（语法糖）

组件的生命周期

  1. 创建
    • getDefaultProps
  2. 实例化
    • getInitialState
    • componentWillMount
    • render
    • componentDidMount -这时候会根据虚拟DOM创建真实DOM，可以再这个方法中读取DOM节点
  3. 更新阶段
    • componentWillReceiveProps
    • shouldComponentUpdate -如果返回值是false，后三个方法不执行
    • componentWillUpdate
    • render
    • componentDidUpdate
  4. 销毁阶段
    • componentWillUnmount
