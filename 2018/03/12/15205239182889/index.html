2018-03-12 | RN

nvm npm node

  1. nvm use v9.7.1 切换使用的node和npm版本（当前shell窗口有用，关了再开就不行了）
  2. nvm list 查看当前电脑上的node所有版本
  3. nvm install v4.7.2 安装指定版本node
  4. 如果是brew安装的node就直接用brew uninstall node就行了
  5. 切换默认的版本nvm alias default v9.7.1
