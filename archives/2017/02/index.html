hexo部署和更新博客

有时候会出现hexo server不能使用的情况，这是因为hexo3中server被分离出来，需要先sudo npm install hexo-server,或者检查一下server模块是不是安装在博客目录下，当前是不是在博客目录下 安装自动部署工具$ npm install hexo-deployer-git --save 更改hexo服务器端口号$ hexo server -p 5000，从默认的4000更改到5000 修改
