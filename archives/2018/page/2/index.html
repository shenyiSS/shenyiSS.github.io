git学习笔记

集中式版本控制：CVS，SVN，分布式：git git的优势是在强大的分支管理 git add 添加工作区的文件到暂存区 git commit 把暂存区的所有更改提交更改到仓库 git status 是用来查看当前仓库的状态 git diff 是用来查看工作区和暂存区的比较，git diff --cached是比较暂存区和分支的比较 git log 是用来查看提交日志的，可以添加--pretty=oneline可以简化日志显示 git
