Swift 中的 Any Anyobject optional

Any 和 Anyobject因为编译的时候不会对 id 类型的变量进行类型检查，可以表示任意实例变量。在 swift 中相对应的是 Anyobject，为了对 cocoa 框架中很多方法做出兼容。因为 id 可以指向 nil，所以一个 API 的返回值为 id 的时候，对应的 swift 里面返回的是 Anyobject?。 Any 可以指向 class，struct 和 enum 等任意类型，Anyobject 只能指向 class
